# Gomoku-AI-Minimax

Play a game of gomoku on Python's tkinter UI.
You will play against an AI that will find its optimal move using minimax algorithm with alpha-beta pruning.

## Setup

1. Make sure Python 3 is installed
2. Run the following

```shell
python main.py
```
